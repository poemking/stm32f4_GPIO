stm32f4_GPIO
============

LCD (Liquid Crystal Display)
------------------------------

Specification ：http://www.sdec.com.tw/products.php?no=50


It has **14 pin** to be used.



- LCD pin 1 = ground

- LCD pin 2 = v+

- LCD pin 3 = ground (or use variable resistor)

- LCD pin 4 ~ 13 connection ：

  - RS  = PD.08 (PIN 8 of GPIOD)

  - R/W = PD.09

  - E   =  PD.10

  - DB0 = PD.00

  - DB1 = PD.01

  - DB2 = PD.02

  - DB3 = PD.03

  - DB4 = PD.04

  - DB5 = PD.05

  - DB6 = PD.06

  - DB7 = PD.07
